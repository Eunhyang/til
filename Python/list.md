#### 리스트 연산자

<u>리스트 더하기</u> : a + b

````
>>> a = [1, 2, 3]
>>> b = [4, 5, 6]
>>> a + b
[1, 2, 3, 4, 5, 6]
````

**<u>리스트 반복</u>** : a * 3

````
>>> a = [1, 2, 3]
>>> a * 3
[1, 2, 3, 1, 2, 3, 1, 2, 3]
````



#### 리스트 수정, 변경, 삭제

<u>리스트에서 연속된 범위의 값 수정하기</u> : a[1:2] = ['a', 'b', 'c']

````
>>> a = [1, 2, 3]
>>> a[1:2] = ['a', 'b', 'c']
>>> a
[1, 'a', 'b', 'c', 4]
````

<u>[] 사용해 리스트 요소 삭제</u> : a[1:3] = [ ]

````
>>> a
[1, 'a', 'b', 'c', 4]
>>> a[1:3] = [ ]
>>> a
[1, 'c', 4]
````

<u>del 함수 사용해서 삭제</u> : del a[1]

````
>>> a
[1, 'c', 4]
>>> del a[1]
>>> a
[1, 4]
````



#### 리스트 관련 함수

- append : 리스트 요소 추가
- sort : 리스트 순서대로 정렬
  - 숫자순 or 알파벳 순
- reverse : 리스트 뒤집기
- index : 위치 반환
- insert : 요소 삽입
  - insert(a, b)는 리스트의 a번째 위치에 b를 삽입하는 함수
- remove : 리스트 요소 제거
  - remove(x)는 리스트에서 첫 번째로 나오는 x를 삭제
- pop : 리스트 요소 끄집어내기
  - pop()은 리스트의 맨 마지막 요소를 돌려 주고 그 요소는 삭제
- count(x) : 리스트에 포함된 요소 x의 개수 세기
- extend(x) : x에는 리스트만 올 수 있으며 원래의 a 리스트에 x 리스트를 더하게 된다.
