## Viewport

viewport는 스마트기기 상에서, 최초에 페이지를 로딩할 때 확대정도, 최대, 최소 확대비율등을 다루는 meta data에 속하는 속성

#### 사용법

`<meta name="viewport" content="" />`를 head사이에 추가

- content 값

| 속성    | 값                      | 초기값         | 특징                         |
| ----- | ---------------------- | ----------- | -------------------------- |
| width | <viewport-length>{1,2} | 독자적 속성에 따름. | device-width 라는 값을 넣을 수 있음 |
|       |                        |             |                            |
|       |                        |             |                            |
|       |                        |             |                            |
|       |                        |             |                            |
|       |                        |             |                            |
